GetWindowDecorations()

Returns the non-client decoration sizes of a standard window on the current system: border width (left/right) and title-bar height (top).

Declaration

Returns
Type Description
NativeMethods.WindowDecorations
Remarks

BorderWidth = SM_CXFRAME + SM_CXPADDEDBORDER TitleBarHeight = SM_CYCAPTION + SM_CYFRAME + SM_CXPADDEDBORDER

You can then offset your MoveAndResize by these values to place the *client* area exactly at screen (0,0).

MoveAndResize(IntPtr?, int?, int?, int?, int?)

Attempts to move and/or resize the specified window.
Any parameter left null preserves its current value.

Declaration
public static bool MoveAndResize(IntPtr? hWnd, int? x = null, int? y = null, int? width = null, int? height = null)
Parameters
Type Name Description
IntPtr? hWnd

Handle of the window to move/resize.
int? x

New left coordinate in pixels; if null, keeps the existing X.
int? y

New top coordinate in pixels; if null, keeps the existing Y.
int? width

New width in pixels; if null, keeps the existing width.
int? height

New height in pixels; if null, keeps the existing height.
Returns
Type Description
bool

true if the window was moved/resized successfully; otherwise false.
Examples
// Move only:
bool moved = WindowManager.MoveAndResize(hwnd, x: 0, y: 0);

// Resize only:
bool resized = WindowManager.MoveAndResize(hwnd, width: 800, height: 600);

// Move and resize:
bool ok = WindowManager.MoveAndResize(hwnd, 50, 50, 1024, 768);
