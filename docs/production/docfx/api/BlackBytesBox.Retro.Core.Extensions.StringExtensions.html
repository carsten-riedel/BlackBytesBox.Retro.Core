Parameters
Type Name Description

The string to trim.
string trimChars

Returns
Type Description
string

A new string with the specified characters removed from the end.
Remarks

Uses ordinal comparison to remove any character contained in trimChars from the end.

Examples
var result = "data>>>".TrimEnd(">");
// result == "data"
