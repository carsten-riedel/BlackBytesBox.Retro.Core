EnsureConsole()

Ensures the process has a console, then redirects Console.In, Console.Out, and Console.Error to it.

Declaration
public static void EnsureConsole()
Remarks

First attempts AttachConsole(–1); if that fails, calls AllocConsole().

Then for each of STDIN/STDOUT/STDERR:

  • Retrieves the native handle
  • Checks it isn’t INVALID
  • Creates the appropriate StreamReader/Writer
  • Calls Console.SetIn/Out/Error

Any failure in redirecting one stream will be caught and logged, but won’t prevent attempts on the others.

Examples
ConsoleManager.EnsureConsole();
Console.WriteLine("This is standard output");
Console.Error.WriteLine("This is standard error");
string line = Console.ReadLine();

SetFont(string)

Attempts to set the legacy console’s font to the specified TrueType face.

Declaration
public static void SetFont(string faceName)
Parameters
Type Name Description
string faceName

The exact font name as registered under HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Console\TrueTypeFont (e.g. “Noto Sans Mono CJK JP”). That font must be installed, fixed-pitch, and TrueType.
Remarks

Only fonts listed under the “TrueTypeFont” registry key and marked fixed-width TrueType can be selected. You must add your desired CJK or emoji-capable font there beforehand.

This API cannot register fonts in the registry for you—it only switches among those already registered.

Examples
ConsoleManager.EnsureConsole();
ConsoleManager.SetFont("Noto Sans Mono CJK JP");
ConsoleManager.EnableUtf8();
Console.WriteLine("漢 😊");
Exceptions
Type Condition
InvalidOperationException

Thrown if the console font cannot be set. The message contains the Win32 error code.
